Um den Code anzupassen, damit er eine Vorhersage für jeden Tag in den nächsten 30 Tagen macht, müssen Sie die Schleife um den Vorhersageschritt hinzufügen. Hier ist der angepasste Code:

```python
# Importieren der erforderlichen Klassen und Funktionen
from statsmodels.tsa.arima.model import ARIMA
from statsmodels.tsa.statespace.sarimax import SARIMAX

# Build Model
model = SARIMAX(train_data, order=(1, 1, 0))
fitted = model.fit(disp=-1)

# Anzahl der Tage für die Vorhersage
forecast_days = 30

# Vorhersage für jeden Tag in den nächsten 30 Tagen machen
predictions = []
for day in range(forecast_days):
    # Vorhersage für den nächsten Tag machen
    fc = fitted.forecast(1, alpha=0.05)  # 95% Konfidenz
    prediction = fc[0][0]  # Vorhersagewert
    predictions.append(prediction)
    
    # Den vorhergesagten Wert zur trainierten Datenreihe hinzufügen, um die Vorhersage für den nächsten Tag zu verbessern
    train_data = train_data.append(pd.Series([prediction], index=[test_data.index[day]]))
    
    # Das Modell mit der erweiterten trainierten Datenreihe neu anpassen
    model = SARIMAX(train_data, order=(1, 1, 0))
    fitted = model.fit(disp=-1)
    
# Den vorhergesagten Zeitraum erstellen
forecast_index = pd.date_range(start=test_data.index[-1] + pd.DateOffset(1), periods=forecast_days, freq='D')
fc_series = pd.Series(predictions, index=forecast_index)

# Plot
plt.figure(figsize=(10, 5), dpi=100)
plt.plot(train_data, label='Trainingsdaten')
plt.plot(test_data, color='blue', label='Tatsächliche Aktienkurse')
plt.plot(fc_series, color='orange', label='Vorhergesagte Aktienkurse')
plt.title('ARCH CAPITAL GROUP Aktienkursvorhersage')
plt.xlabel('Zeit')
plt.ylabel('ARCH CAPITAL GROUP Aktienkurs')
plt.legend(loc='upper left', fontsize=8)
plt.show()
```

Dieser Code fügt den vorhergesagten Wert für den nächsten Tag zur trainierten Datenreihe hinzu und passt das Modell mit den aktualisierten Daten an, bevor er die Vorhersage für den nächsten Tag macht. Dieser Vorgang wird für 30 Tage wiederholt, und die Vorhersagewerte werden in einer Liste gespeichert und schließlich geplottet.
